an empty quoted token had started. It therefore dropped `""`. The other six
held-out cases passed.

`schema-migration` copied only the outer dictionary. Unmodified nested values
remained shared with the input, so mutating a nested list in the returned
document also mutated the original. A deep copy or equivalent structural
copying was required. Six of seven held-out cases passed.

deterministic heap ordering. It assigned graph keys an order before scanning
dependency-only nodes, so `build` was selected before the earlier dependency
`sign`. Six of seven held-out cases passed.
